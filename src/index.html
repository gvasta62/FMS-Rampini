Come funziona

I file candump (N).csv contengono i frame CAN grezzi del canale FMS (J1939). La dashboard li decodifica usando la "chiave" del file DBC del costruttore (RAMPINI_ELTRON_TERNI_corretto.dbc, 61 messaggi / 171 segnali) e ne ricava le grandezze fisiche.

  • Match dei frame per PGN (tipo di messaggio).
  • PGN proprietari 0xFF00/01/02: valido solo il source 30 (BMS).
  • Decodifica J1939: Intel/Motorola, segno, scala, offset, tabelle valori, multiplexing.
  • Grandezza derivata Potenza istantanea = Vpack × IPack da BMS_STAT1 (~4 Hz).

Nota: la posizione GPS non transita sul canale FMS; i contatori energetici di BMS_ENERGY possono restare a 0 perché non popolati dal BMS (vedi documenti in docs/).
